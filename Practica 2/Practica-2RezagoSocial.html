Introducción

Esta nota contiene dos tipos de análisis: Exploratorio y confirmatorio. El primero solamente se usa cuando el modelo de medición está pobre o limitadamente especificado.

Datos

Para esta práctica vamos a trabajar con datos del índice de rezago social del CONEVAL 2015. Originalmente este índice municipal buscaba compensar por la falta de cifras de pobreza a nivel municipal. Hoy en día es parte del paquete de indicadores que le da el CONEVAL a la cámara de diputados para informar el PEF.

Datos en stata.

RS<-read_dta("RezagoSocial2015.dta")

Confiabilidad Omega (\(\omega\))

Recuerden que hay dos formas o estrategias de análisis de datos: Exploratoria y Confirmatoria. Hemos hablado que la confirmatoria es el ideal en medición pero en ocasiones no contamos con el conocimiento o teoría para lanzarnos a calcular un modelo confirmatorio. A continuación exploraremos ambos caminos paara los mismos datos. Esto es para propósitos ilustrativos. Recuerden que no son formas complementarias sino sustitutas de análisis de datos.

Omega exploratorio

Para calcular \(\omega\) a partir de un modelo factorial exploratorio utilizaremos el paquete psych y la función omega(). Esta función calcula \(\omega\) a partir de un modelo factorial exploratorio asumiendo una estrutura de tres dimensiones (el default en las opciones es una estructura de tres dimensiones), en este caso asumiremos que no hay dimensiones puesto que partimos de que la medición es unidimensional. Para ello specificamos nfactors=1.

Para obtener la varianza total pueden usar omega3$schmid$sl.

  • g: Carga factorial del factor general
  • F1-F3: Carga factorial del factor de segundo orden
  • h2: La varianza total que se explica por los factores. \(\lambda_g^2 + \lambda_f^2\)
  • u2: El error total del indicador
  • p2: EL % de la varianza comun que es del factor general. Fuerza del factor general. .97>.12
